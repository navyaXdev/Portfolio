Who is Dinesh?

Hey — I'm Dinesh Patra, a first-year student with a genuine obsession for how systems break and how to defend them. While most people my year are still learning "Hello World", I'm poking around Linux internals, running network scans, and cracking Wi-Fi labs in a controlled environment.

I started with Python and C for the fundamentals, picked up web tech (HTML, CSS, JS) to understand the attack surface, and now I live in the terminal — Bash, nmap, iwconfig, netstat — it's my natural habitat.

The end game? Professional Web Penetration Testing. Every skill I'm building right now is a step on that path.
